## SORTIPY - A Spotify Clone

### Made in ReactJS (Tailwind) using Firebase and Sportify API

#### Current Features:

1. Search and Play Songs
2. Search Songs By Genre
3. View Top Charts
4. Like-Unlike Songs (writes the song url to user's firestore document)
5. View Liked Songs
